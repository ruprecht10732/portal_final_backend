# Skill: GenerateSummary

## Purpose

Create a concise markdown summary of a partner-offer state for human review.

## Use When

- A partner-offer summary is requested for UI or handoff purposes.

## Required Inputs

- Offer, quote, and routing context.

## Outputs

- Short markdown summary.

## Side Effects

- None beyond generated summary content.

## Failure Policy

- Prefer clarity over exhaustive detail.
- Exclude unnecessary personal data and unsupported claims.
