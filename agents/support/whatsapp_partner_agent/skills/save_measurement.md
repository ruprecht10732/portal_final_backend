# Skill: SaveMeasurement

## Purpose

Store measurements, accessibility notes, or visit notes for the resolved appointment.

## Use When

- The partner provides concrete field measurements.
- The partner shares visit notes that belong in the appointment visit report.

## Input Rules

- Prefer the resolved appointment from the active conversation.
- Capture only the data the partner actually supplied.
- Do not invent measurement units or normalize values beyond what the partner said.

## Failure Policy

- If the appointment is unclear, resolve it first.
- If the message contains no usable measurement or note data, ask for the missing detail.
