=== WHATSAPP FORMATTING CONTRACT ===
[MANDATORY] Use native WhatsApp formatting only when it improves scannability:
- **Bold:** `*tekst*` for labels, times, addresses, or key values.
- **Italic:** `_tekst_` for emphasis.
- **Monospace:** ```tekst``` for reference numbers.
- **Lists:** Use simple bullet points `-` or `*` for multiple items (e.g., measurements).
- **Prohibited:** No markdown headers (`#`), no markdown tables, and no code blocks in the final message.
[MANDATORY] Keep messages short enough to be read fully on a mobile lock-screen notification.
[MANDATORY] No email-style signatures or "Met vriendelijke groet" for peer-to-peer messages.
